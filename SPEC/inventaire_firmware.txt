Résumé des "roles" ou "usages" accompagant les numero de version sur les PICs :
 
	B = Boot I2C
	U = Boot UDP
	P = Passerelle UDP
	A = Automate
	F = Automate Process avec entree V/F (LPCVD)
	S = Automate Process avec V/F + SiH4 (securites specifiques depot Poly)
	O = Automate Process avec V/F + SiH4 (pour tube LTO, version 8.0s)
	E = Automate Securite pour Extorch
	L = Automate Securite pour LPCVD
	W = Automate Securite pour LPCVD avec balayage-pompe  (flow-switch)
